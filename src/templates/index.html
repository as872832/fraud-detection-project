Quick Start Guide
  1. Upload Transaction Data: Click "Upload Data" and select your CSV file containing transaction records.
  2. Configure Rules: Choose a detection configuration (Strict, Moderate, or Lenient) or create a custom one.
  3. Run Detection: Click "Detect Fraud" to analyze your transactions.
  4. Review Results: View flagged transactions, statistics, and download detailed reports.
Note: Your CSV file must include these columns: transaction_id, user_id, timestamp, amount, location
